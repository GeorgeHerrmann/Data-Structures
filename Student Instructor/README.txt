The program allows users to login as a student or instructor and view specific data from text files.

To compile the program:
    - Type make into the console
To run the program
    - Type ./main [INSTRUCTORS FILE] [STUDENTS FILE] into the console
        - Example: ./main instructors.txt students.txt
To clean the exectuable
    - Type make clean into the console
